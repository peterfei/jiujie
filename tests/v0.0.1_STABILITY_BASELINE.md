# 九界：渡劫 v0.0.1 核心稳定性基线 (2026-01-30)

## 1. 3D 渲染 (Graphics) 基线
### 角色立牌 (Combatant Sprites)
*   **可见性**: 已通过补全 `Visibility` 组件链确保角色 3D 实体现身。
*   **高度**: 默认 Y 轴高度设为 `0.8`，彻底消除与地板法阵（Y: -1.5）的深度冲突。
*   **材质**: 统一使用 `AlphaMode::Blend` + `unlit: true`，确保跨平台（Windows DirectX / macOS Metal）视觉一致性。
*   **比例**: 修行者尺寸校准为 `120x140`，无拉伸。

## 2. 音频系统 (Audio) 基线
### SFX 稳定性
*   **规范**: 统一 `.ogg` 格式。
*   **健壮性**: 撤销了 brittle 的物理路径检查，回归 Bevy 官方 `AssetServer` 异步加载模式，适配 macOS App Bundle 工作目录。

## 3. 打包与发布 (Distribution) 基线
### 多平台识别
*   **Windows**: EXE 图标嵌入成功，支持 MSI 安装包。
*   **macOS**: 自动构建标准 `.app` Bundle，图标自动识别。
*   **CI**: GitHub Actions 自动化流水线已锁定。

## 4. 逻辑加固
*   **无阻塞 IO**: 所有磁盘操作（存档/读档）均已通过 `commands.queue` 离群化，杜绝 UI 假死。
*   **初始化保护**: 地图进度资源在进入状态时自动补全，解决了“重新开始”闪退。

## 5. 测试覆盖情况
*   核心回归集: 80% 自动化通过 (除环境依赖项外)
*   压力测试: ✓ 通过
