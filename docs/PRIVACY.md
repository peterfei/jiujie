# 隐私策略 (Privacy Policy)

**生效日期：** 2026年2月1日

感谢您游玩《九界：渡劫》(Jiujie: Tribulation)。本隐私策略旨在说明我们在您使用本游戏时如何处理您的信息。

## 1. 信息收集 (Information Collection)

**我们不收集任何个人数据。**

《九界：渡劫》是一款单机游戏。所有的游戏进度、存档、设置和日志文件均存储在您的本地设备上。我们不会将任何数据上传至我们的服务器或第三方服务器。

## 2. 本地数据 (Local Data)

游戏会在您的设备上生成以下文件以确保正常运行：
*   **存档文件：** 用于保存您的游戏进度。
*   **日志文件：** 用于记录游戏运行时的错误信息（仅用于本地调试）。
*   **配置文件：** 用于保存您的画面、声音等设置偏好。

您可以随时在本地文件系统中删除这些数据。

## 3. 网络权限 (Network Permissions)

虽然游戏可能请求网络权限，但这通常仅用于：
*   验证应用商店的购买许可（如 Microsoft Store DRM）。
*   本机系统的必要API调用。

游戏本身不会主动连接互联网发送您的个人信息。

## 4. 儿童隐私 (Children's Privacy)

既然我们不收集任何数据，我们也自然不会有意收集 13 岁以下儿童的个人信息。

## 5. 联系我们 (Contact Us)

如果您对本隐私策略有任何疑问，请通过以下方式联系我们：
*   GitHub Issues: https://github.com/peterfei/JiuJie/issues

# Privacy Policy

**Effective Date:** February 1, 2026

Thank you for playing "Jiujie: Tribulation". This Privacy Policy describes how we handle your information when you use our game.

## 1. Information Collection

**We do not collect any personal data.**

"Jiujie: Tribulation" is a standalone, offline game. All game progress, save files, settings, and log files are stored locally on your device. We do not upload any data to our servers or third-party servers.

## 2. Local Data

The game generates the following files on your device to function correctly:
*   **Save Files:** To store your game progress.
*   **Log Files:** To record runtime errors for local debugging purposes.
*   **Config Files:** To save your graphics and audio preferences.

You can delete these data from your local file system at any time.

## 3. Network Permissions

While the game may request network permissions, this is typically only used for:
*   Validating app store licenses (e.g., Microsoft Store DRM).
*   Essential OS API calls.

The game itself does not actively connect to the internet to send your personal information.

## 4. Children's Privacy

Since we do not collect any data, we knowingly do not collect personal information from children under the age of 13.

## 5. Contact Us

If you have any questions about this Privacy Policy, please contact us via:
*   GitHub Issues: https://github.com/peterfei/JiuJie/issues
